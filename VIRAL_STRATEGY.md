# 🚀 바이럴 마케팅 & API 비용 절감 전략

## 💡 핵심 아이디어

**문제**: Tavily API는 월 1,000회만 무료
**해결**: 생성된 월드컵을 DB에 저장하고 재활용 → **API 사용 90% 절감**

## 🔄 작동 방식

### 1. 첫 생성 (Tavily API 사용)

```
사용자: "2024 K-POP 여자 아이돌" 입력
   ↓
Tavily API 호출 (1회 차감)
   ↓
16명 후보 수집 + 이미지
   ↓
Supabase에 저장 (ID: abc123)
   ↓
URL: /game?id=abc123
```

### 2. 공유 링크 클릭 (API 사용 안함!)

```
친구: URL 클릭 (/game?id=abc123)
   ↓
Supabase에서 불러오기 (무료!)
   ↓
동일한 16명 후보로 게임 시작
   ↓
재생 횟수 +1
```

**결과**: 1번 생성으로 무한 재생 가능! 🎉

## 🎯 바이럴 루프 설계

### A. 결과 공유 버튼

```typescript
// game/page.tsx - 결과 페이지
<button onClick={handleShare}>
  친구에게 공유하기
</button>
```

**공유 시 URL:**
```
https://ai-worldcup.vercel.app/game?id=abc123

제목: "2024 K-POP 여자 아이돌 - 내 선택은 장원영! 🏆"
```

**효과**:
- 친구가 클릭 → 동일한 월드컵 재생
- 그 친구도 공유 → 또 다른 친구 유입
- **무한 루프**! 📈

### B. 인기 월드컵 목록

```typescript
// page.tsx - 메인 페이지
🔥 인기 월드컵
- 2024 K-POP 여자 아이돌 (1,234회)
- 서울 가성비 오마카세 (987회)
- ...
```

**효과**:
1. 신규 방문자가 즉시 재생 가능
2. Tavily API 사용 안함
3. 트래픽 집중 → 더 많은 공유

### C. 트렌딩 주제 추천

```typescript
// 인기 주제 자동 노출
const trendingTopics = [

];
```

**효과**:
- 클릭 시 기존 월드컵 재활용
- 새로 만들 필요 없음
